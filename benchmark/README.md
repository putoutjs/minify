# Benchmark

This is a performance benchmark of the following bundlers:

- Bun
- esbuild
- Terser
- @putoutjs/minify

To run the benchmark:

- Need to install bun globally (https://bun.sh/docs/installation)

```sh
$ npm install
$ node index.js
```
